# Tour Guide Globe with AI Chat

Welcome to the Tour Guide Globe project! This application leverages AI chat capabilities to provide users with an interactive and personalized travel guide experience.

## Features

- **AI-Powered Chat**: Get instant travel advice, recommendations, and answers to your questions.
- **Interactive Globe**: Explore destinations visually on a 3D globe.
- **Personalized Suggestions**: Tailored recommendations based on user preferences.
- **Multilingual Support**: Communicate in multiple languages for a global experience.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tour-guide-globe.git
   ```
2. Navigate to the project directory:
   ```bash
   cd tour-guide-globe
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000`.

## Technologies Used

- **Frontend**: React, Three.js
- **Backend**: Node.js, Express
- **AI Integration**: OpenAI API
- **Styling**: Tailwind CSS
